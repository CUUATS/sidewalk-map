Sidewalk Explorer by CUUATS About

Welcome to Sidewalk Explorer, an interactive viewer for the Sidewalk Network Inventory and Assessment. The inventory and assessment process is an effort to create a comprehensive database of sidewalk network features within the Champaign Urbana Urbanized Area. The database is designed to track the condition and Americans with Disabilities Act (ADA) compliance of the sidewalk network.

Initial data collection took place between May 2014 and August 2015. The database is updated each summer with features constructed or modified during the previous calendar year. Field staff examine and record measurements for all sidewalks, curb ramps, crosswalks, and pedestrian signals in the public right-of-way.

The results of the inventory are analyzed using an ADA compliance index that scores each variable on a scale from 0 to 100. A score of 100 represents preliminary compliance with the proposed Public Right-of-Way Accessibility Guidelines (PROWAG). A similar index rates condition for sidewalks and curb ramps.

The assessment scores are one of several factors local agencies use in prioritizing sidewalk network improvements. For a discussion of other factors influencing prioritization, download the report document.
